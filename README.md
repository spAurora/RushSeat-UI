 # 武大图书馆抢座程序RushSeat-UI(R1.0.1)<br>
 注意：软件已过期<br>
- - -
## 目前实现的功能：<br>

后台监测某个房间的座位，有释放的自动预约 <br>
抢座成功后发送提示短信（已过期）<br>
针对信图某一房间抢各种类型(靠窗、有电脑)座位<br>
检索某个房间某个时间段的所有空座并预约<br>
加入倒计时，22:45准时抢座<br>
抢指定座位<br>
抢指定座位失败后自动改签附近座位<br>
急速抢座模式<br>
抢座后自动关机<br>
保存账号密码（DES加密）<br>
拓展检索空间（检索房间组合)<br>
